## General for most scripts
| Function Name | Purpose |
|---------------|---------|
| loaddata(filepath) | Load from file (txt, json, csv, etc.) |
| add_element(structure, element) | Insert new item |
| remove_element(structure, element) | Delete item |
| update_element(structure, old, new) | Modify item |
| search_element(structure, query) | Check existence |
| print_structure(structure) | Display contents |
| clear_structure(structure) | Empty the structure |
| copy_structure(structure) | Return a copy |
| merge_structures(structure1, structure2) | Combine two structures |
| main() | Demo usage of all above |
